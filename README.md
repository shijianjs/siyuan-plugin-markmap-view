
# Siyuan markmap view plugin

[中文版](./README_zh_CN.md)


## Operations
- Install the plugin
- An icon will appear in the top right toolbar
- Open a document
- Click the icon to pop up the markmap view
    - Or use the shortcut key `Ctrl + Alt + M`
